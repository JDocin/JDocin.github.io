2019北邮计算机803考研经验分享（1）【基础篇】

置顶 | 发表于 2019-07-11 | 分类于 考研 | | 浏览 次

写在最前面

​ 北邮信息安全专业，三年综合排名前50%，19年报考本校网络技术研究院学硕（计算机科学与技术），初试成绩380（数学一122，计算机综合119，英语一77，政治62），复试成绩80，综合成绩报考组第一，已录取。

​ 其实这个经验分享好久前就写好了，给了几个关系好的学弟学妹，最近试着搭了自己的博客，想了想还是打算发出来，也算是对自己经历过的事情的一个记录吧。主要从基础阶段，强化阶段，冲刺阶段，报考和复试几个方面来谈，下面接入主题。

基础篇

​ 对于本校生来说，我觉得开始时间不用特别早，大概在3月份左右就可以。也就是说大三开学的时候，就应该正式进入每天定量的复习了。但是据我观察数学应该是每年越来越难的（毕竟考生越来越多），如果觉得自己数学基础一般也可以提前学，怎么学后面再说。英语单词也是尽早背后期压力比较小。

​ 初期阶段是 【3月(或更早)–暑假开始前】这个阶段，主要进行大面积的数学复习，英语基础打底（背单词）的复习。

① 数学复习方法

​ 1-高数：这个阶段的数学我建议你直接听汤家凤老师的数学基础班(若2020还没出，听19的也没关系，数学每年大纲都没啥变动)。汤老师认真负责尤其是基础差的，可以说是手把手教你怎么做题那种，跟着听课记笔记就可以(基础班的高数一定要跟着汤老师做笔记啊！)。最好一次性(1-2天，如果时间不够3天也可以)听完一章，比如第一章极限，然后用1-2天把练习册对应的习题做了，一定要重视自己动手做题的阶段！什么练习册一会说。这样大概一周以内完成一章的学习的速度进行。

​ 2-线代：全程抱紧李永乐大腿就行。也是这个阶段先听基础班，然后做题。听课可以在笔记本上补充他这个薄讲义没有的东西，不用全部记录。

​ 3-概率论：有人推王式安，但我听不下去他讲课，你可以试试，或者听张宇的基础班也行，汤老的概率论有点尴尬，不聊了。套路还是一样的。注意线代和概率论的题没有高数的多，做题时间可以相对缩短，而且可能这个时候到了期末，概率论后期那些东西可以适当跳过(尤其是数字特征以后的东西看不懂的先跳过，也别做题，但是数字特征部分还是很重要的要重视)。

​ 4-书目推荐：高数买一本张宇18讲(不是张宇带你学，别买错了)，线代买一本李永乐线性代数就足够，概率论买一本张宇的概率论9讲。这三本都是属于半参考书性质的，有知识公式例题，当然习题可能量比较少。

​ 我建议补充一本张宇的1000题或者汤家凤的1800题，我买的是1800，太多了没做完，我认为如果听汤老的课，可以做宇哥的题，这样覆盖范围更广。线性代数一定以李永乐的那本讲义为主，再做其他的！(1000题可以只做A B组，c组的太难，基础阶段可以不管)，如果有精力我建议再加一本复习全书，这本书的高数和概率论部分的习题都非常经典，可以在强化阶段做。

② 英语复习方法

​ 我英语基础差（相对年纪其他同学来说），四级510左右？六级490+低空飘过。正因为我基础差，所以花了挺多时间在英语上的，最后成绩出来我还是挺满意的毕竟北京区（不是王婆卖瓜，因为真的花了蛮多时间，逃~）。

​ 基础这个阶段我建议你就踏踏实实背单词，攻克长难句。单词不用去听课纯属浪费时间，买一本单词书，红宝书或者何凯文1575，讲道理何凯文虽然是个渣渣但是这本单词书涵盖了很多真题的单词，挺棒的，每天背1个list，注意及时复习，争取在一个半月内背诵完一遍，然后再来一遍，争取一个月内完成。没错，单词不在乎你看了多少，而在于你是否反复记住了。背单词一定要耐心，每天分出2个小时来，不要急躁，不要觉得没用，考研是必须要求词汇量的，只要坚持就一定会有成果。

​ 这里要强调一下：我同期有好几个同学，因为英语基础不好，或者说觉得英语分值小，采取了放弃英语，专攻数学专业课的方法。我认识的人里，这样的同学或者是总分上了330，但是英语最后没过国家线（国家线是底线，就算打了400分，只要有一科不过线也白搭），或者是整体垮掉，英语低其他也低。几乎没有人险胜。所以我劝有这样想法的学弟学妹不要投机取巧，不要轻易放弃，踏踏实实复习，只要你付出的够多，最后肯定会有满意的分数。

​ 回归正题，背了两遍单词（或3遍）之后就可以开始学长难句和阅读的基础，我建议听唐叔的语法基础班，大概只有6 7节课，每天听一节好好消化。不要做模拟的阅读题，因为那和真题套路相差甚远。

​ 如果这个也听完了还没到暑假，我建议可以听一听唐静老师的翻译强化班，没错直接听强化，讲义自己打印一份，因为翻译本身就是长难句，可以帮助你夯实阅读基础。我建议听完被动结构翻译成主动那块结束就行了，足够了。也可以不听这个，做97-2003年的真题阅读，每天一篇，好好看解析。或者两个穿插着，翻译为主阅读为辅，看你时间。那个完型先不用管。
​ 注意，这个的时候也要每天复习单词。

小结

​ 基础阶段就是这样啦，这个阶段我没有看专业课和政治，跨考不算。因为北邮的专业课虽然多，但是试卷难度还可以，如果基础差也可以提前看看数据结构，前提是你数学学的很好，很踏实进度也跟得上。具体的专业课复习知识写在强化和冲刺篇里。政治不用着急，相信肖大大（逃~。

​ 这个阶段就是要沉得住气，好好学数学和英语，锻炼计算能力理解概念，夯实词汇量。切记和别人攀比速度，只要完成自己阶段性目标就好。

​ 另外，如果下定决心考研（比如我，连保研的边也沾不到），就不要那么在乎这学期的成绩、排名啦。尽量选写论文的课，水课，能水就水，不要挂科就好了。因为不出国或者保研的话这个成绩都是没什么特别重要的。人总要学会取舍的嘛，初试成绩高才是硬道理，别的都是浮云。
