2019北邮计算机803考研经验分享（2）【强化篇】

发表于 2019-07-12 | 分类于 考研 | | 浏览 次

前言

​ 从暑假开始(期末结束后大概7月)，考研正式进入强化阶段，持续到差不多10月份。这个阶段也是决定你最终结果最重要的阶段，必须要认真对待。

​ 对于暑假要不要回家的问题，我觉得回家待2周之内是可以接受的，但是你必须保证自己可以自律学习。即可以每天按时起床不赖床，起来后就开始学习，不能累了就倒下眯一会玩一会电脑看一会电视那种，晚上不熬夜看小说，不出去与朋友聚会，不去亲朋好友应酬等等，如果你说我差不多能做到那你就回去，但是时间最好不要超过两周。如果你觉得你不能，那就心里有点数，还是老老实实就在学校学习吧。

数学强化

​ 这个暑假是数学综合能力提升的关键时期！！！必须认真对待。

​ 高数和概率论你就听张宇的强化班，张宇的基础班有一点水，但是强化班真的不错。这个时候2020的应该已经出了，听最新的。讲义自己打印一份（各种公众号里都有），认认真真学。同样的，听完一章去把对应习题做了。这个时候可以做复习全书（虽然很厚但是题型真的蛮经典）还有张宇1000里面你剩下的ab组，复习错题也很重要（做错很多次的题建议重点标注，有代表性的可以整理错题本，冲刺阶段很有用），c组我只能说选做吧没时间不做也没毛病。无论做什么，都要吃透及时整理笔记和错题。

​ 线代还是跟着李永乐就行，注意复习全书的题是他讲课的时候参考的习题，所以可以做张宇的题。

​ 暑假有全天的时间复习，所以速度比基础班要更快了。至少在开学之前，高数强化应该全部结束，或者高数+线代强化全部结束。(如果基础班结束的早，那就提前开始强化班，我说的时间点都是最晚开始的时间点)。

​ 数学这边，我建议自己能构建知识框架（看一遍书，合上之后自己想题型和做法等）。就比如高数后面的各种积分的几何意义、常值意义、特殊奇偶性和公式等；比如线代特征值有什么题型，矩阵有什么题型；尤其是概率论，题型很固定。自己总结在笔记上，再标出对应的例题，最后复习的时候看看简直清爽的八达鸟。（想看我整理的可以留个邮箱，但是估计开学才能发，暑假搬宿舍书已经封箱了= =）。

​ 这个阶段要夯实解题的基本功，多做难题复杂题，一定要自己上手，不要觉得自己会了就不做了！提升做数学题的手感。

英语

​ 这个阶段可以开始做阅读了(英语直接买一套张剑的黄皮书真题就完事。之前如果没做97-03可以从00年开始做，节省时间，每天1篇，最开始可以不用定时，看懂即可，错太多也不要紧别慌这都是正常的，我做97的时候错了8、9个。最多2周时间搞定这些老年份的阅读，兼顾背单词。

​ 当你做到04年的时候，接下来唐叔唐迟要闪亮登场了，他真的是一个很厉害的老师阅读讲的贼好。在b站上找他的阅读，注意区分英语一和英语二，应该会找到一个2004–2017的合集，就听这个就行。这时也不能忘记重复背之前的单词哦。

​ 这个阶段前期，每天做1/2篇阅读，然后听唐叔讲课，前几篇还会涉及到讲几个题型，你做好笔记，每个题型留一篇纸那种(方便以后整理错题)。

​ 另外，要注意整理真题单词，大概从2010开始，近10年真题你要熟背，年份越近越重要，因为单词重复出现率真的很高！跟着唐叔，踏踏实实的做，一定要自己先做再听课！不要着急，每天一篇就行。等做到了13年，建议开始成套模考，就是4篇阅读一起做，但是最后要留3年的题来考前模拟，这个后期冲刺会再说。一定要重视真题单词。

​ 04往后的真题建议弄两套，因为到了后期肯定要再做一次，我个别的年份做了三次。如果不怕麻烦整个铅笔也行，但还是最好弄两份，打印就行别怕麻烦。英语开始学习阅读之后，一定不要间断做题，手感是很重要的，我记得我英语巅峰时期应该是11月初，当时做整套的卷子完型错0-2个阅读错0-2个，但是后来因为忙于复习专业课和数学就间断了后来手感就没有那么好了。如果后期卷子都做了2遍实在没题，就买英语二的做。

​ 完型和新题型这个阶段不用管，翻译跟着阅读练。

​ 暑假我放了大块时间在数学上，英语有点耽搁了，只是在背单词。大概开学的时候才开始从04年开始跟着唐叔做，其实也来得及。另外对于何凯文和唐叔的争议，我建议，趁着时间早可以都听听，选择适合自己的。但是一旦确定下来就不要再改变了，要相信这个老师坚定的跟着他走，调整自己的思维去适应真题的思路。

政治

​ 政治开学再开始其实就可以了，担心的话就8月中旬往后开始。我建议前期不用花太多时间，每天1-1.5个小时足够，买肖秀荣的1000题和精讲精练或者徐涛的那本任选，听徐涛老师的课就足够，肖大大讲课太催眠了hhh，听完课刷一下对应的1000题就完事。

​ 哲学刚开始比较难，也是考察重点，要耐心的学下去，踏踏实实做题理解。到了史纲我觉得就是背，因为没啥需要理解的，跟着徐涛快速过一遍就OK。

​ 然后思修法基我觉得其实不用听课的。。。。因为分值小的可怜而且题型就那么几个，做一下1000题总结一下笔记就好。而且后期的肖八肖四几乎都会涉及，你就自己看看书就行，如果不是想特别高分其实这部分1000题也不用做可以先放着，节约时间。

​ 到了毛中特就比较重要，要好好听课，认真背，认真做题！尤其是五位一体和四个全面那里最为重点还有前面的新民主主义那里也常考。

专业课

​ 北邮计算机803专业课有4门，我从9月开学正式开始复习的，感觉后期时间有些吃紧，可以从8月开始比较好（但是暑假仍然以数学为主，看数学进度来安排其他科目）。

数据结构

​ 虽然学的时候可能觉得难但是数据结构确实是4门里最简单的了。书籍的话建议单独买一本天勤的数据结构然后其他买王道的。不需要看原教材，时间根本来不及。就踏踏实实看没什么特别的。争取1-2天结束一章的知识点加习题。

​ 说几个考察重点，数组、链表、队列、栈、字符串、二叉树、图、排序、查找（诶一不小心全涵盖了？？）因为都是考核范围。前面四个很可能会出手写代码的大题，因为是最简单的。字符串、排序、查找选择题居多，尤其是排序查找各种特性（复杂度等）很容易考。树是重中之重，不多说了设计到树的处处是重点！图也很重要选择题多，若是了代码题估计考场上没几个写得出。

​ 计组，操作系统，计网不想展开说了，他们三个的辅导书用王道别用天勤。老老实实学就行，争取1-2天过一章（知识点+习题）。碰到难的地方，比如计组芯片那里，还有数据结构字符串匹配那个算法，如果到了第三天还没看懂就跳过，因为四门科目知识点真的爆炸，抠一个超难的点性价比太低。但是不能一直一直跳哈，有困难还是需要客服的。课后习题以选择题为主，大题随缘，时间够就做，不够就放着。大题的提升阶段在后期，后面再说。
