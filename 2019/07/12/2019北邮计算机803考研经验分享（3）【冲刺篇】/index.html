2019北邮计算机803考研经验分享（3）【冲刺篇】

发表于 2019-07-12 | 分类于 考研 | | 浏览 次

数学

​ 大约10月份国庆开始，着手做真题。真题书买张宇的那个真题大全解，写的很好，答案的形式也很好。我建议做真题的时候用一张活页纸或者白纸当答题卡，把答案写上面，正规一点，也方便第二次再做和错题查询。

​ 前期速度慢一点从97开始就可以，卷子是肯定够的。两天或三天完成一篇，第一天做，第二天第三天整体错题和总结。给自己规定时间，一上午(争取3小时内完成)，98 99比较难，做的时候心里别慌踏踏实实写就行。97-04的做的时候可以翻翻书，但是后面我建议还是独立完成，不要不会了就看书看答案，当然一些考前背一背喝前摇一摇的公式还是可以查的。

​ 前期发现哪里有缺漏，赶紧第二天去看一看这里的知识点，做一些这里之前的错题之类的，及时巩固。到后面08-15就提升自己的速度，一天写一个，必须在3小时内完成。

​ 2010年后的真题要反复做，之前的真题要再过一遍错题，如果有做的特别不好的年份也可以做一遍。数学和英语真题都是要做2+的，所以不要刻意去记或者回想答案。不会的地方及时补缺，重要的是，数学做题手感很重要，隔三天不做题那水平就会大打折扣，所以一定不要中断这个过程。

​ 做多了真题就会发现，概率论、线代还有高数的多重积分那里题型非常固定，这样前面提过的总结框架的方法就派上用场了，一定要善于总结和反思，虽然多做题也很好但是吃死题海战术也不是一个很聪明的方法。

​ 最后留3套卷，考前一周模考。 建议留17 18 19。

英语

整套模考

​ 基础阶段阅读（04-12/13）都做的差不多了就可以开始冲刺阶段（12/13年开始整套模考），其实应该能感觉得自己思维能力的提升和方式的转变。整套题的做题顺序，我的建议是阅读-新题型-翻译-完型。给自己规定时间最好在2小时内完成以上的内容。写完了一起对答案。先不写作文，最后留2套完整卷子，考前几天做做训练手感。

​ 一定要重视10年内的真题单词。大概11月份开始有计划的开始学新题型，这个不难唐叔有大概2小时的课，听那个就行了，然后把04以后的新题型刷2遍，最后复习复习就行。

​ 完型不用刻意的练，跟着大阅读做就行，你会发现阅读水平上了之后，你的语感会变得很好，完型水平自然就上来了。做个几年的真题就行，另外整套卷子里，最后做完型，你懂的。

​ 翻译后期没有时间就跟着整套卷子练一下，没办法还是得有取舍。

作文

​ 当13-17年的卷子做完后，你就可以开始学作文了，买本王江涛的高分写作，听一下他的2020救命班或者强化班，跟着他走就完事你大爷还是你大爷。跟着背作文，前期你会觉得难，但是跟后期政治一比，英语作文不算事儿。救命班应该会给压5-6篇作文，你要是不放心，就早点开始背，把那10篇他给压得大作文都背了，不用背模拟的，足够了。小作文也是先听听他讲的，各种形式的背个一两篇，小作文没法押题主要是看发挥了，大作文水平上来了小作文绝对没毛病。

​ 模板我特别不建议，这几年查的很严，套模板会比你写一堆简单句分数还低，不多说了。

​ 我开始的比较晚，大概10月中旬，当时背了6篇大作文（必须滚瓜烂熟，每天早上晚上各背一遍），2篇小作文（时间实在不够了）。结果你猜怎么着，两篇作文真的被我押上了。考场上翻开作文差点笑出声，整个状态就上来了，迅速的写完作文然后开始开心的做阅读（最后的顺序，可以先4篇阅读再作文，也可以先作文，看自己的习惯）。

​ 有人会质疑，背作文真的有用吗？我想说其实你背的不只是作文，而是人家的写作思路和表达方式。就拿19大作文来说，虽然题目是没背过的，但是写作思路（如何引出主题，对比讨论）、与主题相近的句子、经典的句型蹭蹭的往出蹦，最后我写作文的过程变成了一个挑选好句子替换主题词的过程。

​ 具体的还是要看王江涛的课，总之一句话，多背多记跟着道长不迷路。另外在背作文的阶段要做以前的真题阅读，阅读不能停，如果所有04-17的都做过2-3次了，实在没题做，你可以买本英语二。英语二本质上来说和文章和英语一难度相似，思路一致，只是提问简单了而已，所以可以拿来练手，主要是不能断了阅读英文文章。于此同时，单词尤其是近10年真题单词也不能放掉，时间还是很紧的，自己要合理安排。

政治

​ 跟紧肖秀荣不迷路！如果听课结束的早我建议你再做一遍1000题里马原的多选题(先多选有时间再单选)，毛中特五位一体四个全面新民主那的多选题，买一本徐涛背诵笔记，背史纲的各种东西，不要浮躁，这一阶段把选择题底子打牢。安心等待肖四肖八出版。

​ 肖八的大题没时间可以先不管，把选择题出现的知识点，注意不是答案，而是涉及的知识点，自己整理出来，最好把对应1000题里的错题也看看，把关键的一起整理一下，马原毛中特史纲思修分开整理。整理知识点有助于减轻后期背诵压力。

​ 肖四大概在考试前10多天出来，肖四选择大多数是肖八原题，可以用来模考，肖四的大题必须滚瓜烂熟，后几天必须空出时间每天背几个小时政治！你大爷还是你大爷不服不行，另外后期不要心慌，每年都有人阴谋论还不是考前紧张考后真香。

​ 政治后期心态会很慌，但是我还是要说一句就好好总结1000题的选择和肖八肖四的选择知识点、错题，考前拿出来背背。还有肖四，大题一定要滚瓜烂熟，及时复习。上60还是没问题的，北京地区，政治上60就算正常水平不会拉分了。

专业课

​ 四本书都看完后，开始冲刺阶段，差不多11月之前开始。买一本王道的408真题。一定要重视真题，最开始做说不定会崩溃，我做第一篇的那天晚上就失眠了，因为真的不会做，选择题错了快一半，大题根本没写字。

​ 我想说，这都是正常的，别慌。后来我找了一个学长聊了聊，他劝我不要着急，不会的就翻书查，实在不会就看解析，但是，一定要看懂解析。这个阶段总结就十分十分重要。因为你没有时间去再温习一遍书籍，只能靠题去帮你复习，刷题为王。前期2-3天一套。先做题，不确定的选择题或者错题的涉及到的知识点的相关章节，一定要仔细的看，然后总结在笔记上。相近的题型总结到大题。

​ 大题前两套基本是不会做的，没关系，看完题干理解一下就看解析，看懂为止。看懂后合上书，再看题干自己做一遍。做了两套之后你会发现大题的题型很局限，所以一定要总结相似题型！！！到了后面的卷子就先自己思考试着做一下，然后看答案看解析总结。

​ 大题真的都是有套路的，比如计组内存cache那里可能会与虚拟内存即页那里相结合来出题，如果不理解就会蒙圈，对于这一类问题就要牢牢把握存储容量和地址位数的关系。还有计网里面算IP，选择路由器，算子网之类的，都是有套路的。

​ 最后，如果你能顺利做出408的大题或者做出一半，那你就没问题了。因为最后你做803真题就知道了，是真尼玛简单。如果408是S++那么803就是A–。所以说不要慌，就按照题目来覆盖知识点，好好总结选择题，规整大题的知识点，没问题的。803有些题选择题比较狗，但是没关系，努力刷题，攻克难题，总结类似题，随便打个110+还是稳妥的。110应该算还可以的分数，大部分人集中在100左右，130算很高分了一般这个分数的人总分都390+了，这个分布应该懂得不多说了。

​ 考前2周，找803真题来做，可以问一下自己的学长学姐或者北邮人论坛上应该也有。或者评论里留个邮箱，我要是能找到的话就发，但是没有19年的。这个真题都是没十分正确的答案的，做来练练手就好，不必较真儿。

总结

​ 学习相关的就更到这里了，有不明白的可以在评论里留言我会及时回复大家的。接下来可能会更一些考前心态调整/心路历程、以及报考的相关建议之类的。

​ 暑假正值考研的攻坚期，大家要努力复习，同时也要兼顾身体健康。预祝大家复习顺利，取得自己满意的分数。

​ 最后，欢迎报考北京邮电大学，See U In BUPT！
